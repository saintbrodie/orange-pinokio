# Orange Pinokio Launcher

1-click launcher for [Orange](https://github.com/saintbrodie/Orange), a minimalist, dynamic web frontend wrapper around **ComfyUI**.

## Usage
- **Install**: Clones the repo and installs local environment.
- **Start**: Launches the frontend server.
- **Update**: Updates both the Pinokio launcher and the Orange repository.
- **Reset**: Removes the cloned repo and local environment for a clean start.

## Requirements
- Pinokio
- A running instance of ComfyUI (running at its standard port).
